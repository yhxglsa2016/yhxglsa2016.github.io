JavaScript学习之旅

第一节JS的简介

什么是语言：纸袋机：机器语言 汇编语言：符号语言 现代语言：高级语言
JavaScript诞生于1995年 由网景公司发明 最初起名LiveScript后来由于SUN公司接入更名为JavaScript 作用：用来处理网页中的前端验证（检查用户输入内容是否符号一定规范 用户名长度 密码长度 邮箱格式）
JS的标准名命名为ECMAScript ECMAScript是一个标准，而这个标准需要由各个厂商去实现
一个完整的JavaScript由ECMAScript DOM BOM三个部分构成

特点：解释型语言 类似于C和Java的语法结构 动态语言 基于原型的面向对象

第二节 JS的入门小程序HelloWorld

<script></script>的默认样式为<script type="text/javascript"></script>

三条输出语句：

控制浏览器弹出一个警告框alert("内容XXX");
使计算机在页面中输出一个内容document.write("内容XXX") (内容出现在HTML中的body)
向浏览器的控制台输出内容console.log("内容XXX")(控制台在浏览器F12开发者工具中)

JS代码是从上到下依次执行

第三节 JS的编写位置

JS代码编写到标签的onclick属性中 当我们点击按钮时，js代码才会执行（onclick是属性）
<button onclick="alert('讨厌，你点我干嘛~~');">点我一下</button>
注意事项：单双引号需注意嵌套规范

JS代码写在超链接的href属性中，这样当点击超链接时，会执行js代码
<a href="javascript:alert('让你点你就点！！');">你也点我一下</a>`` `` <a href="javascript:;">你也点我一下</a>

JS代码可以写在HTML标签属性中，但是它们属于结构与行为耦合，不方便维护，不推荐使用
JS代码编写到Script标签中 JS代码编写在外部JS文件中 然后通过Script标签引入
优点：写在外部文件中可以方便在不同的页面中同时引入，也可以利用浏览器的缓存机制（推荐使用）
引入格式<script src="../JavaScript/03.js"></script>
Script标签一旦用于引入外部文件后，就不能使用当前标签编写代码(每一个Script标签只能编写一个代码) 如果需要可以继续创建新的Script标签进行编写代码

第四节 JS的基本语法

语句格式:严格区分大小写JavaScript是严格区分大小写的，abc与Abc在编译器中也会被认为是两个不同的东西
每一条语句以分号（;）结尾 如果不写分号，浏览器会自动添加，会消耗一部分资源，浏览器在某些时候会加错分号，所以开发中分号必须写
JS中忽略多个空格和换行，所以我们可以使用空格和换行对代码进行格式化

注释：注释中的内容不会被解析器执行，但是会在源码中进行显示，一般用来对程序中的内容进行解释
注释格式：单行注释：//注释内容（Ctrl+/） 多行注释：/*注释内容*/（Shift+Alt+A）

第五节 字面量和变量

字面量：一些不可改变的量 字面量都是可以直接使用 但一般不会直接使用字面量（如1 2 3 4 5 6）

变量：可以用来保存字面量，而且变量的值可以随意改变 变量更加方便我们的使用，所以在开发中可以通过变量去保存一个字面量，而很少直接使用字面量
声明变量：在JS中使用var关键字在声明一个变量var a;
赋值变量：为变量赋值a = 123;
声明与赋值变量：同时进行var b = 789
注：通过变量对字面量进行描述

第六节 标识符

标识符：在JS中所有可以由我们自助命名的都可以称为标识符（例如：变量，函数，属性的名字或函数的参数）
标识符格式：第一个字符必须是字母，下划线（_），或者美元符号（$） 其他字符可以由字母，下划线，美元符号或者数字构成
注意：按照惯例ECMAScript标识符需要采用驼峰命名法（首字母小写，每个单词的开头字母大写，其余字母小写） 但是需要注意JavaScript中的标识符不能是关键字和保留字符
JS底层保存标识符时实际上采用Unicode编码 所以理论上，所有的utf-8中含有的内容都可以作为标识符（中文也在utf-8中，但不推荐使用中文作为标识符）

第八节 强制类型转换

指将一个数据类型强制转换为其他的数据类型 类型转换主要指，将其他的数据类型，转换为String Number Boolean

将其他数据类型转换为String：

方式一：
调用被转换数据类型的toString()方法 调用xxx的yyy()方法就是xxx.yyy()
该方法不会影响到原变量，它会将转换的结果返回
注意：null和undefined这两个值没有toString()方法，如果调用它们的方法，会报错

方式二：
调用String()函数，并将被转换的数据作为参数传递给函数 调用xxx()函数就是xxx()
使用String()函数做强制类型转换时，
对于Number和Boolean实际上就是调用的toString()方法
对于null和undefined，就不会调用toString()方法 它会将 null 直接转换为 “null”将 undefined 直接转换为 “undefined”

将其他数据类型转换为Number：

方法一：
使用Number()函数

true 转成 1 false 转成 0

第九节 其它进制的数字

在JS中，如果需要表示进制的数字需要前缀：16进制0x 8进制0 2进制0b
注意：不是所有的浏览器都支持2进制 如久远的IE 但目前主流浏览器都已支持
像”070”这种字符串，有些浏览器会当成8进制解析，有些会当成10进制解析，所以输出结果也会不同
解决方案：可以在parseInt()中传递一个第二个参数，来指定数字的进制（底数）如a = parseInt(a,10);
注意：如果提供了十六进制参数，那么字符串前面的”0x”可以省掉
parseFloat()只解析十进制值，因此不能指定底数

第十节 运算符

运算符也叫操作符 通过运算符可以对一个或多个值进行运算并获取运算结果
比如typeof就是运算符，可以用来获取一个值的类型 它会将该值的类型以字符串的形式返回（number string boolean undefined object）

算数运算符：
+（两个值加法运算）对非Number的值会先转化为数字在进行运算 布尔值的运算 true为1 false为0 任何值和NaN做运算结果都是NaN 字符串相加为拼串（两个字符串拼接为一个字符串） 任何值和字符串做加法运算，会先转化为字符串，在于字符串进行拼串
隐式转化：将任意的数据类型+“”即可将其转换为String var c = 123 c = c+""
使结果变明显：（“c= ”+c）;

-（两个值减法运算）
（两个值乘法运算）
/（两个值除法运算）
任何值做- /运算时都会自动转换为Number
隐式转化：可以通过一个值 -0 *1 /1来将其转化为Number %（两个值取模运算）

一元运算符：
一元运算符，只需要一个操作数 + 正号 不会对数字产生影响 对其他数据类型使用+使其转化成Number - 负号 会对数字产生影响 对于非Number的值先转换成Number

自增与自减：
自增 ++ 通过自增可以使变量在自身的基础上增加1 对于一个变量自增后原变量的值会立即自增1
自增分成两种：后++(a++) 和 前++(++a) 无论是a++ 还是 ++a，都会立即使原变量的值自增1
不同的是a++ 和 ++a的值不同 a++的值等于原变量的值（自增前的值） ++a的值等于新值 （自增后的值）
自减 — 通过自减可以使变量在自身的基础上减少1 对于一个变量自减以后，原变量的值会立即自减1
自减分成两种：后—(a—) 和 前—(—a) 无论是a— 还是 —a 都会立即使原变量的值自减1
不同的是a— 和 —a的值不同 a— 是变量的原值 （自减前的值）—a 是变量的新值 （自减以后的值）

每日重点：强制转化的意义格式与方法 自增与自减的理解 运算时的顺序

逻辑运算符：
JS中有三种逻辑运算符
！ 非 可以用来对一个值进行非运算（对一个布尔值进行取反操作 true变falsa falsa变true）若对一个值进行两次取反，值不会进行变化 对非布尔值进行运算，则会将其转化为布尔值，然后取反（利用该特点可以将其他数据类型转化为布尔值 当其去两次返时，将其转化为布尔值）
&& 与 可以用来对符号两侧的值进行与运算并返回结果 true && true结果为true true && false结果为false false && false结果为false
总结：两个值中只要有false则与运算的结果返回false JS中的“与”属于短路的与，如果第一个值是false，则不会检查第二个值 例子：true && alert("不会出来！！"); false && alert("我就出来！！");
|| 或 可以用来对符号两侧的值进行或运算并返回结果 false || false;结果为false true || false;结果为true false || true;结果为true true || true;结果为true
总结：两个值中只要有true就返回结果为true JS中的“或”属于短路的或，如果第一个值是true，则不会检查第二个值 例子：true || alert("不会出来！！"); false || alert("我就出来！！");

与和或的非布尔值的情况：先将其转化为布尔值再进行运算，并且返回原值
与运算：若两个值都为true，则返回后边的 若两个值中有false，则返回靠前的false值 即若第一个值为true，则必然返回第二个值 若第一个值为false，则必然返回第一个值
或运算：若第一个值为true，则直接返回第一个值 若第一个值为false，则直接返回第二个值

赋值运算符：
= 可以将符号右侧的值赋值给符号左边的变量
+= a += 5 等价于 a = a + 5
-= a -= 5 等价于 a = a - 5
*= a -= 5 等价于 a = a - 5
/= a /= 5 等价于 a = a / 5
%= a %= 5 等价于 a = a % 5

关系运算符：
通过关系运算符可以比较两个值之间的大小关系 如果关系成立则会返回true 关系不成立则返回false

                >    大于号    判断符号左侧的值大于右侧的值    
                >=  大于等于    判断符号左侧的值大于等于右侧的值
                <    小于号    判断符号左侧的值小于右侧的值
                <=  小于等于    判断符号左侧的值小于等于右侧的值

​ 非数值的情况：会将其转化为数字然后进行比较 任何值与NaN做比较都是false 如果符号两侧的值都是字符串不会将其转化为数字进行比较而是分别比较字符串中字符的Unicode编码（比较字符编码时是一位一位进行比较 如果两位一样，则比较下一位，所以借用它来对英文进行排序 比较中文时没有意义）
​ 注意：如果比较的两个字符串型的数字，可能会得到不可预期的结果 在比较两个字符串型的数字时，一定一定一定要转型

相等运算符：
== 相等运算符用来比较两个值是否相等 相等则返回true否则返回false 当使用==来比较两个值时如果值的类型不同则会自动进行类型转化成相同的类型再进行比较

​ undefined 衍生自 null 在做相等判断时会返回true
​ NaN不和任何值相等包括它本身 可以通过isNaN()函数来判断一个值是否是NaN 如果该值时NaN则返回true否则返回false console.log(isNaN(a))

​ ！= 不相等用来判断两个值是否不相等 如果不相等则返回true 否则返回true 如果比较两个值不相等时值的类型不同则会进行自动的类型转换再进行比较

​ === 全等用来判断两个值是否全等 它和相等类似 不同的是它不会做类型的自动转换 如果两个值的类型不同则直接返回false

​ ！== 不全等用来判断两个值是否不全等 它和不相等类似 不同的是它不会做类型的自动转换 如果两个值的类型不同则直接返回true

条件运算符（三元运算符）：
语法： 条件表达式?语句1:语句2
执行的流程：条件运算符在执行时，首先对条件运算符进行求值 如果该值为true，则执行语句1，并返回结果 如果该值为false，则执行语句2，并返回结果 如果条件的表达式的求值结果是一个非布尔值会将其转换为布尔值然后再进行运算
